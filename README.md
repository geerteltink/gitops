# GitOps

## Develop

Run:

```bash
$ go run main.go
```

Compile and run:

```bash
$ go build .
$ .\gitops.exe
```

Test:

```bash
$ go test ./...
$ go test -coverage -coverprofile=coverage.out ./...
$ go tool cover -html=coverage.out -o coverage.html
```
